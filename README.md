# Cabinet Bogdan Georgescu — site de prezentare

Site de prezentare (one-page) pentru un cabinet de psihoterapie psihanalitică.
Construit ca fișier static: `index.html` + folderul `images/`.

## Rulare locală
Deschide `index.html` în browser, sau servește folderul:

```
npx serve cabinet-bogdan-georgescu
```

## Structură
- `index.html` — pagina (HTML + CSS + JS inline)
- `images/` — logo, portret, foto cabinet și restul resurselor

## De completat (date de la client)
- Poze în rezoluție mare (portret + cabinet)
- E-mail, program, adresă exactă (hartă)
- Rețele sociale, răspunsuri FAQ, tarif online
- Pagini legale (confidențialitate / termeni)
- Domeniu + găzduire
